# Dashboard Overview


The dashboard provides a high-level view of engineering health across four primary metric areas.

## Key Dashboard Metrics

### 1. Top 3 Invested Repositories

Shows the repositories with the highest engineering investment — useful for understanding where the most time and effort is being directed.

### 2. PR Cycle Time

Displays the average PR cycle time and its trend over time. Helps identify bottlenecks in the code review and merge process.

### 3. Code Stability

Provides insights into code stability, including:
- Release durability
- Percentage of engineering time dedicated to new value development

### 4. GenAI Adoption

Shows the adoption rate of GenAI tools and the activities they impact across the engineering organization.
