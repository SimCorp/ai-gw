# Engineering Investment


Engineering Investment analytics help teams understand resource allocation and identify optimization opportunities.

## Features

| Feature | Source | Access | Description | Actionable Insight |
|---|---|---|---|---|
| Engineering Effort Overview | Git | ✅ | Total engineering effort measured in effort units or $ spent | Ensure alignment with priorities |
| Wasted Spend / Non-Prod Work | Git | ✅ | Percentage of work spent on code idle >3 months, not deployed to production | Reallocate effort away from waste |
| Investment by Work Type | PM + Git | ✅ | Breakdown of engineering effort across key work categories | Balance time according to org needs |
| Teams/Groups Work Time per Category | PM + Git | 🔒 | Compare engineering effort across work types and teams/groups | Validate effort distribution across org |
| Non-Prod Work Analysis | PM + Git | 🔒 | Non-prod work per team/group across different effort types | Ensure efficient engineering investment |
| Feature Spend by Issue Type | PM + Git | 🔒 | Development effort by initiatives and per issue | Focus effort on what matters most |
| Feature and Issue-Level Coding Time vs. Estimated | PM + Git | 🔒 | Actual vs. planned effort at feature/issue level | Refine estimation model |

**Access levels:** ✅ = Open access · 🔒 = Restricted access
