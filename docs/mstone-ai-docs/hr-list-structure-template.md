# HR List Structure Template


The HR List Structure Template aligns your organizational hierarchy with the engineering analytics platform so that data can be attributed to the right teams and groups.

## Template Fields

The template contains a single tab with the following columns:

| Field | Required | Description |
|---|---|---|
| Developer Name | Yes | Full name |
| Developer Email | Yes | Work email address |
| Team Name | Yes | The developer's team assignment |
| Group Name | Yes | Higher-level organizational unit |
| Site Name | No | Location or organizational site |
| Git Username | No | GitHub/Git username |

## How to Fill It In

- Add **one row per developer**
- Use **consistent naming** for teams, groups, and sites across all rows
- Leave optional fields blank if the information is unavailable
- No strict formatting requirements — use your organization's standard terminology

## After Completion

Send the completed file to your **onboarding contact**. They will integrate it with Git, Jira, and GenAI data to create an engineering organization overview within the platform.
