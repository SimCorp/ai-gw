# Roles


The platform supports flexible user access levels with five distinct roles.

## User Roles

| Role | Permissions |
|---|---|
| **Admin** | Manage integrations, roles, and configurations |
| **Team Lead (TL)** | View and customize dashboards for their assigned teams and directly supervised developers |
| **Group Manager (GRM/GM)** | View and customize dashboards within their managed groups and associated teams |
| **User** | View and customize dashboards |
| **Viewer** | Shareable view-only mode — does not consume a seat license |
