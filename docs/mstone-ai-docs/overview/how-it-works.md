# How It Works


The platform integrates with tools like **GitHub**, **Jira**, and **Cursor** to process metadata and generate performance dashboards.

## Focus Areas

The platform processes data across three primary areas:

- **Engineering performance metrics** — PR cycle time, code stability, developer productivity
- **Repository investment analysis** — where engineering time is allocated and what it produces
- **AI-assisted development metrics** — adoption, engagement, and ROI of GenAI tools

## Deployment Options

The platform can be deployed in two ways — both provide identical feature access:

| Model | Description |
|---|---|
| **SaaS** | Hosted externally; no infrastructure management required |
| **On-Premises** | Self-hosted on your own infrastructure; full control over data residency |

Organizations choose based on their security, compliance, and data sovereignty requirements.
