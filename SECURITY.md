# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities **privately** — do not open a public issue.

Use GitHub's private vulnerability reporting: go to the repository's **Security** tab →
**Report a vulnerability** (or
[**this link**](https://github.com/SimCorp/ai-gw/security/advisories/new)). This opens a
private advisory visible only to the maintainers.

We aim to acknowledge new reports within a few business days and will keep you updated as we
investigate and prepare a fix.

## Scope

This repository is the SimCorp AI Gateway. Reports concerning the gateway services
(`services/*`), the developer/admin portals, and the deployment/CI configuration in this repo
are in scope. Vulnerabilities in third-party dependencies are tracked separately via Dependabot.

## Supported versions

This is an internal platform deployed from `master`; security fixes are applied to `master`.
